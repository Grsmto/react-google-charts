# React Google Charts

<img src={logoUrl} alt="Logo" className="logo" />

React Google Charts is a lightweight and fully typed React wrapper for [Google Charts](https://developers.google.com/chart/interactive/docs/reference).

It's easy to use, supports over 25 chart types, and is highly customizable when needed.

- **Lightweight and Typed**: A thin layer over Google Charts with TypeScript support for type safety.
- **25+ Chart Types**: Access a diverse set of charts like Line, Bar, Pie, Gantt, Timeline, Sankey and more.
- **Flexible API**: Easily switch between chart types and customize them to suit your needs.
- **Interactive and Animated**: Enhance user engagement with interactive features and animations.
- **Easy Integration**: Simple to set up and integrate into your existing React projects.
- **Complete Customization**: Full control to customize every aspect of your charts, from colors to labels and tooltips.

Start creating dynamic and interactive charts with minimal effort using React Google Charts.

## Quickstart

Install this library with your favorite package manager:

```bash
npm install --save react-google-charts
```

Then, import and use it:

```jsx
import { Chart } from "react-google-charts";

<Chart
  // Required
  chartType="ScatterChart"
  data={[
    ["Age", "Weight"],
    [4, 5.5],
    [8, 12],
  ]}
  // Optional
  width="100%"
  height="400px"
  legendToggle
/>;
```
